# Cline Settings

Configuration for [Cline](https://github.com/cline/cline) (VSCode AI coding assistant).

## Installation

```bash
# Copy to your project
cp -r cline-settings/.cline ./

# Or link globally
ln -sf ~/.frankx-ai-settings/cline-settings ~/.cline
```

## Features

- Anthropic Claude as default provider
- Auto-approve safe file types
- FrankX creative assistant persona
- Sensible workspace exclusions

## Custom Instructions

The settings include creator-focused custom instructions that:
- Emphasize technical authority with creative wisdom
- Support the musician-technologist workflow
- Prioritize soul-aligned technology
