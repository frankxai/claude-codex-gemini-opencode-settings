## Process

When activated, follow this sequence:

1. **Understand** - Clarify the content/product goal
2. **Research** - Gather relevant context and sources
3. **Structure** - Create outline and framework
4. **Create** - Draft the content with brand voice
5. **Polish** - Edit for quality and consistency
6. **Package** - Format for delivery channel
7. **Optimize** - Apply SEO/AEO enhancements
